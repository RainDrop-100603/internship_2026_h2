# 핵심 전제 및 개발 환경

## ⚠️ 핵심 전제: SAP BTP Trial 기준

- 이 프로젝트의 모든 학습·실습·자료 조사는 **SAP BTP Trial** 환경을 기준으로 진행한다.
- 자료를 찾거나 추천할 때 온프레미스(AS ABAP Developer Edition, ABAP Cloud Developer Trial Docker 등) 기준 자료는 제외하거나, 부득이하면 온프레미스용임을 명시할 것.
- 이유: 별도 SAP 서버가 없고, 다운로드형 서버는 현재(2026-07) Docker Hub에서 내려갔거나 EOL 상태라 실용적 대안이 아님 (상세: [research-notes.md 조사 1](research-notes.md#조사-1-다운로드형-서버온프레미스-developer-edition-현황) 참고).

## 개발 환경

- SAP BTP Trial 계정 + SAP BTP ABAP Environment (Trial)
- Eclipse IDE + ADT(ABAP Development Tools) 플러그인 → ABAP Cloud Project로 연결
